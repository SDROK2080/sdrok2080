⚙️ How to Set Up Your Files

Step 1: Download All Files

1

Download from outputs folder:

  • INDEX.html
  • KMC-UCDL_Complete_Guide.html
  • HOW_IT_RUNS.html
  • KMC-UCDL_n8n_Workflow.json

Step 2: Create a Folder

2

Put all files in the same folder:

MyDocuments/KMC-UCDL/
├── INDEX.html
├── KMC-UCDL_Complete_Guide.html
├── HOW_IT_RUNS.html
└── KMC-UCDL_n8n_Workflow.json

⚠️ All files MUST be in the same directory!

Step 3: Open INDEX.html

3

Start with the landing page:

  • Double-click INDEX.html
  • Or right-click → "Open with" → Browser
  • Or drag into browser window

Step 4: Use Navigation Buttons

4

Click buttons to navigate:

  • From INDEX: Click "Start Setup Guide" or "Learn How It Works"
  • From Setup Guide: Click "← Home" or "How It Runs →"
  • From How It Runs: Click "← Home" or "📖 Setup Guide"

✅ Testing the Links

Test Each Link:

🔗 INDEX.html Links

  • Test 1: Open INDEX.html → Click "🚀 Start Setup Guide" button → Should open Complete Guide
  • Test 2: Open INDEX.html → Click "📚 Learn How It Works" button → Should open How It Runs
  • Test 3: Open INDEX.html → Click "📖 Setup Guide" card → Should open Complete Guide
  • Test 4: Open INDEX.html → Click "🔄 How It Runs" card → Should open How It Runs

🔗 Complete Guide Links

  • Test 1: Open Complete Guide → Click "← Home" → Should return to INDEX
  • Test 2: Open Complete Guide → Click "How It Runs →" → Should open How It Runs
  • Test 3: Complete Guide has internal section buttons → Should scroll to sections

🔗 How It Runs Links

  • Test 1: Open How It Runs → Click "← Home" → Should return to INDEX
  • Test 2: Open How It Runs → Click "📖 Setup Guide" → Should open Complete Guide

❌ If Links Don't Work:

Problem: Most likely files are in different folders
Solution: Move all HTML files to the same folder
Verify: All 4 files should be visible in the same directory
