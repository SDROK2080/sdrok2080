Node Setup Guide

Type: Cron

Method: GET

Model: gpt-4

Type: OpenAI
Model: gpt-4
Chain: Node 3 → Node 4 → Node 5
Prompts: See Prompts tab

Troubleshooting Guide

Issue 1: Workflow Fails to Trigger

Symptoms: No execution at 5:00 AM
Solutions:
• Check Cron trigger: Is it activated?
• Verify timezone: Asia/Kathmandu?
• Test with manual execution first
• Check n8n service status

Issue 2: OpenAI API Fails

Symptoms: Node 3/4/5/7 shows error
Solutions:
• Check API key validity at platform.openai.com
• Verify GPT-4 access enabled
• Check rate limiting (wait 60 seconds)
• Try GPT-3.5-turbo as fallback

Issue 3: Facebook Publishing Fails

Symptoms: Post not appearing
Solutions:
• Check access token expired (regenerate)
• Verify page ID is correct
• Check app has publishing permissions
• Review content for policy violations

Issue 4: Telegram Not Notifying

Symptoms: No message received
Solutions:
• Test bot: Send /start to bot
• Verify bot token is correct
• Check chat ID (has minus sign?)
• Remove special characters from message

Issue 5: Google Sheets Not Logging

Symptoms: No row appended
Solutions:
• Share sheet with service account
• Verify sheet ID correct
• Check header row names match exactly
• Test with Google Sheets API explorer

Issue 6: Image Generation Fails

Symptoms: No image URL returned
Solutions:
• Check Pollinations.ai status
• Simplify prompt (no special chars)
• Use alternative: Stability AI, DALL-E
• Increase timeout to 30 seconds

Issue 7: Content Quality Issues

Symptoms: Generic or repetitive text
Solutions:
• Refine prompts with more examples
• Increase temperature to 0.8-1.0
• Increase max tokens (300+ for Node 5)
• Add human review step before publishing
