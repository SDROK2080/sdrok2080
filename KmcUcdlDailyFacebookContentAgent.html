🔄 How The System Works

Every day at 7:00 AM NST, this happens automatically:

  1. Trigger Fires 🔔 - Time-based scheduler activates
  2. Get Context 📅 - System gets current date, day, time in Nepal timezone
  3. Pick Content Type 🎲 - Randomly selects type based on day of week
  4. Generate Post ✍️ - OpenAI GPT-4 creates unique post (1.9 seconds)
  5. Publish to Facebook 📱 - Posts go live immediately
  6. Log Data 📊 - Entry saved to Google Sheets for tracking
  7. Complete ✅ - Entire process finishes in 3.26 seconds!

Result: Your page has fresh content every morning with zero manual work from you!

📖 SETUP GUIDE - COMPLETE INSTRUCTIONS

🔧 Complete Setup Guide (90 Minutes)

⏱️ Estimated Time: 90 minutes total (one-time setup)

Phase 1: Facebook Setup (15 minutes)

Step 1.1: Create Facebook Developer Account ▼
  1. Visit https://developers.facebook.com
  2. Sign in with your personal Facebook account
  3. Click "Get Started" button
  4. Answer setup questionnaire (2-3 minutes)
  5. Accept terms and conditions
  6. Verify email address if prompted
Step 1.2: Create Your App ▼ 
Path: Developers → My Apps → Create App

Fill in these fields:
- App Name: KMC-UCDL Content Agent
- App Type: Business
- App Purpose: Business
- Contact Email: Your email
- App Description: Daily social media automation for KMC

Select permissions:
- pages_manage_posts ✓
- pages_read_engagement ✓
- pages_manage_metadata ✓
Step 1.3: Get Your Credentials ▼ 
1. Go to Settings > Basic
   - Copy your App ID (save this!)
   - Copy your App Secret (keep private!)

2. Go to Tools > Graph API Explorer
   - Select your app from dropdown
   - Select your Facebook page
   - Click "Get Token"
   - Click "Generate Access Token"
   - Copy the long token (starts with EABC...)
   - This is your FACEBOOK_ACCESS_TOKEN

3. Get your Page ID:
   - Visit your Facebook page
   - Go to About section
   - Look for Page ID (numbers only)
   - This is your FACEBOOK_PAGE_ID

SAVE ALL THREE VALUES!

Phase 2: OpenAI Setup (10 minutes)

Step 2.1: Create OpenAI Account ▼
  1. Visit https://platform.openai.com
  2. Click "Sign up" (use email or Google account)
  3. Verify your email address
  4. Set up 2-factor authentication (recommended)
  5. Complete account setup
Step 2.2: Add Payment Method ▼ 
Path: Left sidebar > Billing > Overview

1. Click "Add payment method"
2. Enter credit or debit card details
3. Complete verification
4. Set monthly spending limit: $10-20 (for safety)

Cost example:
- $0.01 per post × 30 posts = $0.30/month
- Your limit of $20 is very safe!
Step 2.3: Create API Key ▼ 
Path: Left sidebar > API Keys

1. Click "+ Create new secret key"
2. Name it: "KMC-UCDL Facebook Agent"
3. Click "Create secret key"
4. IMPORTANT: Copy the key immediately
   (Format: sk-proj-xxxxxxxxxxxxx)
5. Store it safely - you can't view it again!

This is your OPENAI_API_KEY

Phase 3: n8n Cloud Setup (10 minutes)

Step 3.1: Create n8n Account ▼
  1. Visit https://n8n.cloud
  2. Click "Sign up"
  3. Use email or Google account
  4. Verify email address
  5. Create a workspace named "KMC-UCDL"
  6. Accept terms
Step 3.2: Set Up Credentials ▼ 
In n8n Dashboard:

1. Left sidebar > Credentials > New

2. Create Credential #1: OpenAI
   - Type: OpenAI
   - API Key: [Your sk-proj-... key from OpenAI]
   - Name: "OpenAI API - UCDL"
   - Save

3. Create Credential #2: Google Sheets
   - Type: Google Sheets
   - Click "Connect your Google account"
   - Authorize n8n to access Sheets
   - Name: "Google Sheets - UCDL"
   - Save

4. Create Credential #3: Environment Variables
   - Go to Settings > Environment
   - Add these variables:
     FACEBOOK_PAGE_ID=123456789
     FACEBOOK_ACCESS_TOKEN=EABC...
   - Save

Phase 4: Import Workflow (10 minutes)

Step 4.1: Import n8n Workflow ▼ 
In n8n Cloud:

1. Click "+ New Workflow"
2. Click "..." menu > "Import from file"
3. Select "KMC-UCDL_n8n_Workflow.json"
4. Click "Open"
5. All 9 nodes load into your workflow
6. Click "Save"

Phase 5: Configure Nodes (30 minutes)

Double-click each node and configure:

Node 1: Schedule Trigger
- Trigger Type: Cron
- Cron: "15 1 * * *" (7 AM NST)
- Timezone: Asia/Kathmandu
✅ Save
Node 4: OpenAI (Chat)
- Credential: OpenAI API
- Model: gpt-4-turbo
- Temperature: 0.8
- Max Tokens: 300
✅ Save
Node 6: Facebook POST
- Method: POST
- URL: https://graph.facebook.com/v18.0/[PAGE_ID]/feed
- Headers: Content-Type: application/json
✅ Save
Node 8: Google Sheets
- Credential: Google Sheets
- Spreadsheet: UCDL_Daily_Posts
- Sheet: Sheet1
- Range: A:H
✅ Save

Phase 6: Testing (15 minutes)

1

Test Nodes Individually

Right-click each node > "Execute Node" > Check output

2

Test Full Workflow

Click "Execute Workflow" button at top

3

Verify Facebook

Check your page - post should appear!

4

Verify Google Sheet

Check UCDL_Daily_Posts - new row added?

Phase 7: Activation (2 minutes)

🎉 Final Step!

In n8n, click the "Activate" toggle (top right of workflow)

Status should change to "Active"

Your first automated post will appear tomorrow at 7:00 AM NST!
