📖 SETUP GUIDE

🔧 Complete Setup Guide

Estimated Time: 90 minutes (first time)

Phase 1: Facebook Setup (15 minutes)

Step 1: Create Facebook Developer Account ▼
  1. Go to https://developers.facebook.com
  2. Sign in with your personal Facebook account
  3. Click "Get Started"
  4. Answer setup questions
  5. Accept terms
Step 2: Create Your App ▼ 
Developers → My Apps → Create App

App Name: KMC-UCDL Content Agent
App Type: Business
App Purpose: Business
App Permissions: pages_manage_posts, pages_read_engagement
Step 3: Get Credentials ▼ 
Settings > Basic

Copy and SAVE these:
- App ID: [Your App ID]
- App Secret: [Keep Secret!]

Tools > Graph API Explorer:
- Select your app
- Select your page
- Generate Access Token
- Copy the long token (EABC...)

Phase 2: OpenAI Setup (10 minutes)

Step 1: Create OpenAI Account ▼
  1. Go to https://platform.openai.com
  2. Sign up (email or Google)
  3. Verify email
  4. Set up phone number (2FA)
Step 2: Add Payment Method ▼ 
Left sidebar > Billing > Overview

1. Click "Add payment method"
2. Add credit/debit card
3. Set monthly limit: $10-20 (safe)
Step 3: Create API Key ▼ 
Left sidebar > API Keys

1. Click "+ Create new secret key"
2. Name: "KMC-UCDL Facebook Agent"
3. Copy key (starts with sk-proj-)
4. SAVE IT - you can't see it again!

Cost: ~$0.01 per post × 30 days = $0.30/month

Phase 3: n8n Setup (10 minutes)

Step 1: Create n8n Cloud Account ▼
  1. Go to https://n8n.cloud
  2. Sign up (email or Google)
  3. Verify email
  4. Create workspace (name: "KMC-UCDL")
Step 2: Create Credentials ▼ 
n8n Sidebar > Credentials > New

Create 3 credentials:

A) OpenAI API
   - API Key: sk-proj-...
   - Name: "OpenAI API - UCDL"

B) Google Sheets OAuth2
   - Connect your Google account
   - Name: "Google - UCDL"

C) Environment (in Settings)
   - FACEBOOK_PAGE_ID=123456789
   - FACEBOOK_ACCESS_TOKEN=EABC...

Phase 4: Import & Test

Use the JSON workflow file to import into n8n. Follow setup instructions for node-by-node configuration.

🎉 Setup Complete!

Your workflow is ready. First automated post will appear tomorrow at 7 AM NST.

🔄 HOW IT RUNS

🔄 How KMC-UCDL System Runs

✨ The Simple Version

Every day at 7 AM NST, n8n automatically runs your workflow. It generates unique content, posts to Facebook, and logs everything in ~3 seconds.

🎬 Visual Flow

7:00:00 AM NST
    ↓
[TRIGGER] Schedule fires up ⏱️
    ↓
[CONTEXT] Get date/time (Monday? June 14?)
    ↓
[CATEGORY] Pick random type (Study Tips, etc)
    ↓
[OPENAI] Generate unique post (1.9 sec)
    ↓
[PARSE] Extract text + hashtags
    ↓
[FACEBOOK] Post to page (0.9 sec)
    ↓
[LOG] Save to Google Sheets
    ↓
7:00:03 AM NST ✅ DONE!
