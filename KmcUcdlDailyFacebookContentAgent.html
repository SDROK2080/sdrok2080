📖 Complete Setup Guide (90 Minutes)

⏱️ Total Time Required: 90 minutes (first time only)

📝 What You'll Need: Email account, payment method (optional, $10-20 limit), 3 browser tabs

Phase 1: Facebook Developer Setup (15 minutes)

Step 1: Create Facebook Developer Account ▼
  1. Go to https://developers.facebook.com
  2. Sign in with your personal Facebook account
  3. Click "Get Started" button
  4. Answer setup questions about your business
  5. Accept all terms and conditions
  6. Verify your email if prompted

✅ Success: You should see the Facebook Developers dashboard

Step 2: Create Your Facebook App ▼
  1. In dashboard, click "My Apps" (top right)
  2. Click "Create App" button
  3. Fill in app details:
    • App Name: "KMC-UCDL Content Agent"
    • App Type: Business
    • App Purpose: Business
  4. Accept terms and click "Create App ID"
  5. Complete any security challenges (2FA if needed)

⚠️ Important: Don't close this page. You'll need the App ID next.

Step 3: Add Facebook Page & Get Access Token ▼
  1. In app settings, find "Products" or "Add Product"
  2. Search for and add "Facebook Login" product
  3. Configure your Facebook page:
    • Go to "Tools" → "Graph API Explorer"
    • In dropdown, select your app
    • In next dropdown, select your Facebook page
    • Make sure permissions include: pages_manage_posts, pages_read_engagement
    • Click "Generate Access Token"
  4. Copy the long token (starts with EABC...)
  5. SAVE THIS TOKEN in a safe place - you'll need it for n8n

🤖 Node-by-Node Explanation

Node 1: Schedule Trigger ⏱️

Type: Cron Trigger

When: Every day at 7:00 AM NST

Cron: "15 1 * * *"

Does: Signals workflow to start

Node 2: Get NST Context 📅

Type: Function/Code

Inputs: Current timestamp

Does: Calculates date/time in Nepal timezone

Outputs: dayName, dateEng, isWeekend

Node 3: Select Category 🎲

Type: Function/Code

Inputs: dayOfWeek from Node 2

Does: Random selection based on day

Outputs: category (good_morning, study_tips, etc)

Node 4: OpenAI Generate ✍️

Type: OpenAI API Call

Model: gpt-4o (or gpt-4-turbo)

Does: Generates unique post text

Time: ~1.9 seconds (mostly waiting)

Node 5: Parse Response 📝

Type: Function/Code

Inputs: Raw OpenAI response (JSON)

Does: Extracts text, hashtags, validates

Outputs: finalPost, characterCount

Node 6: POST to Facebook 📱

Type: Facebook Graph API

Endpoint: /feed (your page)

Does: Posts content to your page

Returns: Post ID, timestamp

Node 7: Prepare Log 📋

Type: Function/Code

Inputs: Data from all previous nodes

Does: Formats for Google Sheets

Outputs: Formatted row array

Node 8: Google Sheets 📊

Type: Google Sheets API

Does: Appends row to tracking sheet

Sheet: UCDL_Daily_Posts

Columns: Date, Day, Category, Content, ID, Count, Time, Status

Node 9: Complete ✅

Type: No Operation

Does: Marks workflow as finished

Triggers: Success notification

Total: 3.26 seconds elapsed
