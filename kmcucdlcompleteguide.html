This section explains each of the 9 workflow nodes in detail.

Node 1: Schedule Trigger

Purpose: Trigger workflow every day at scheduled time
Configuration: 
Trigger Type: Cron
Cron Expression: "15 1 * * *" (7 AM NST)
Timezone: Asia/Kathmandu
Common Times (NST → Cron): 
7:00 AM NST  → "15 1 * * *"
8:00 AM NST  → "15 2 * * *"
12:00 PM NST → "15 6 * * *"
5:00 PM NST  → "15 11 * * *"
Help: Visit https://crontab.guru to verify cron expressions

Node 4: OpenAI Content Generation 🔑 IMPORTANT

⚠️ This is the most important node!

Type: OpenAI (Chat)
Must Configure: 
Authentication:
  - Credential: OpenAI API (with your sk-proj-... key)

Model:
  - Select: gpt-4-turbo (or gpt-4o for cheaper)

Temperature: 0.8 (creative but consistent)
Max Tokens: 300
Top P: 0.95

System Message:
  Copy from Content_Generation_Prompt_Library.md
What It Does: Generates unique post text based on day/category
Cost: ~$0.01 per post
Example Output: 
{
  "choices": [{
    "message": {
      "content": "{\"text\": \"Good Morning Warriors!...\"}"
    }
  }]
}

Node 5: Parse AI Response

Purpose: Extract text from OpenAI JSON response
Type: Function (Code)
Configuration: Code is built-in, no changes needed
Output: 
{
  postText: "Good Morning Warriors!...",
  hashtags: "#KMC #UCDL #Success",
  finalPost: "[Combined text + hashtags]",
  characterCount: 215
}

Node 6: Post to Facebook 🔑 CRITICAL

⚠️ This posts to your Facebook page!

Type: HTTP Request
Must Configure: 
Method: POST
URL: https://graph.facebook.com/v18.0/[PAGE_ID]/feed
  (Replace [PAGE_ID] with your page number)

Headers:
  Content-Type: application/json

Body (JSON):
{
  "message": "{{$node['Parse AI Response'].json.finalPost}}",
  "access_token": "{{$env.FACEBOOK_ACCESS_TOKEN}}",
  "published": true
}
Important:
  • Use environment variable for token (secure)
  • published: true = posts immediately
  • published: false = creates draft (for review)
Success Response: 
{
  "id": "123456789_987654321",
  "created_time": "2026-06-14T07:15:00+0000"
}

🐛 Troubleshooting Guide

Something wrong? Follow this systematic troubleshooting guide.

Workflow Won't Trigger

Symptom: Post not appearing at scheduled time ▼
  1. Check Activation: Is toggle "Active"? (top right)
  2. Check Timezone: n8n Settings > Timezone = "Asia/Kathmandu"
  3. Check Cron: Verify at https://crontab.guru
  4. Check Executions: Left sidebar > Executions > Any recent runs?
  5. Manual Test:
    • Disable schedule (click lightning icon)
    • Click "Inject" on Node 1
    • Does workflow run? If yes, schedule issue. If no, node issue.

OpenAI API Error

Symptom: Node 4 shows red error ▼
Error: "Invalid API Key"
  1. Copy-paste error? Go to platform.openai.com/api/keys
  2. Create new key (can't reuse)
  3. Update in n8n credentials
  4. Test Node 4 again
Error: "Quota Exceeded"
  1. Out of budget
  2. Go to openai.com/billing
  3. Add more credits or increase limit
  4. Try again in 1 minute
Error: "Rate Limited"
  1. Too many requests too fast
  2. Normal - will resolve in 1 minute
  3. n8n auto-retries

Facebook Post Fails

Symptom: Node 6 shows error, post doesn't appear ▼
Error: "Invalid Token"
  1. Token expired (happens after 60 days)
  2. Go to developers.facebook.com
  3. Regenerate new token via Graph API Explorer
  4. Update in n8n environment variables
  5. Test Node 6 again
Error: "Page Not Found"
  1. Wrong PAGE_ID
  2. Go to facebook.com/yourpage
  3. Click About > Find Page ID (numbers only)
  4. Update in n8n FACEBOOK_PAGE_ID
  5. Test Node 6
Error: "Insufficient Permissions"
  1. Token missing page_manage_posts permission
  2. Create new System User token (more permanent)
  3. Verify permissions: page_manage_posts ✓
  4. Use that token in n8n

Google Sheet Not Getting Data

Symptom: Workflow succeeds but no data in sheet ▼
  1. Check Sheet Exists:
    • Is it created?
    • Sheet name = "UCDL_Daily_Posts"?
    • Headers in Row 1?
  2. Check Sheet ID:
    • Google Sheet URL: docs.google.com/spreadsheets/d/[ID]/
    • Copy that ID
    • Paste in Node 8 spreadsheetId field
  3. Check Range:
    • Node 8: range = "UCDL_Daily_Posts!A:H"
    • (Adjust sheet name if different)
  4. Check Credentials:
    • Node 8 > Authentication
    • Should show: "Google Sheets OAuth2"
    • With your Google account connected
  5. Test Fix:
    • Reload page (F5)
    • Click Node 8 > Execute Node
    • Google Sheet updates?

Posts Look Wrong / Formatting Issues

Symptom: Emojis missing, text cut off, duplicate hashtags ▼
Issue: Emojis not showing
  • Check Facebook post directly
  • Usually appear fine on Facebook
  • If not, may be device font issue
  • Try refreshing page
Issue: Text cut off
  • Character limit exceeded
  • Check Google Sheet character count column
  • If >250, text being truncated
  • Adjust OpenAI temperature down (0.7)
Issue: Duplicate hashtags
  • OpenAI generating hashtags
  • Also added from prompt
  • Solution: Remove hashtags from system prompt
  • Let OpenAI generate them only
